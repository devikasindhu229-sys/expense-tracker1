# 💰 Expense Tracker

A Flask-based expense tracking web application.

## Features

- User Registration & Login
- Add/Delete Expenses
- Category-wise tracking
- Expense Pie Chart
- Search Expenses
- Date Filter
- Dark Mode UI
- Responsive Design

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run:

python app.py
